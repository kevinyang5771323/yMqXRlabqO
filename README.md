# 健康信息管理系统 python820

本项目是一个基于Python Django框架和MySQL数据库的健康信息管理系统。系统旨在通过网页的形式提供以下功能：

## 项目概述

- 疾病知识科普
- 心理健康测评
- 个人健康数据管理
- 数据可视化分析

## 技术栈

- 后端：Python Django
- 数据库：MySQL
- 前端：HTML, CSS, JavaScript

## 功能模块

### 疾病知识科普

- 展示常见疾病的文章和相关信息

### 心理健康测评

- 提供心理健康测评问卷
- 分析测评结果

### 个人健康数据管理

- 用户注册与登录功能
- 个人健康数据记录

### 数据可视化分析

- 将个人健康数据以图表形式展示

## 系统角色

- 游客：浏览疾病科普文章
- 注册用户：进行心理健康测评、管理个人健康数据

## 运行环境

- Python 3.x
- Django 2.x
- MySQL 5.x

## 部署步骤

1. 安装Python和MySQL环境
2. 下载源码和数据库脚本
3. 导入数据库脚本至MySQL数据库
4. 配置Django项目环境
5. 运行Django项目

## 目录结构

```
project/
│
├── apps/               # 存放Django应用
├── db/                 # 数据库脚本目录
├── media/              # 媒体文件目录
├── static/             # 静态文件目录
├── templates/          # 网页模板目录
├── manage.py           # 项目管理脚本
└── ...
```

## 核心亮点

- 易用的用户界面
- 实时的数据可视化
- 基于Django框架，易于扩展和定制开发
- 完善的个人健康数据管理功能
